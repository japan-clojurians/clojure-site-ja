ClojureインストーラとCLIツール

ClojureはClojure replを起動したり、ClojureやJavaのライブラリを利用したり、Clojureプログラムを起動したりするための コマンドラインツール を提供している。

brew を利用したMacへのインストール

brewを使用してコマンドラインツールをインストールする:

brew install clojure

既にインストールしている場合は次のコマンドで最新のリリースにアップグレードすることができる:

brew upgrade clojure

Linuxへのインストール

  1. 次の依存がインストールされていることを確認する: bash、 curl、 rlwrap、 Java

  2. linux-installスクリプトを利用してダウンロードとインストール実行すると、/usr/local/bin/clj、/usr/local/bin/clojure、および/usr/local/lib/clojureが作成される:

curl -O https://download.clojure.org/install/linux-install-1.9.0.358.sh
chmod +x linux-install-1.9.0.358.sh
sudo ./linux-install-1.9.0.358.sh

プラットフォーム別のインストーラ(apt-getなど)は将来的に提供される。

Windowsへのインストール

現在は提供されていない - LeiningenもしくはBootを参照。

その他のClojureの実行方法

ローカルビルド

ソースをダウンロードし、Clojureをビルドする:

git clone https://github.com/clojure/clojure.git
cd clojure
./antsetup.sh
ant local

そしてローカルjarでREPLを起動する:

java -jar clojure.jar

オンラインでClojureを試す

repl.it はインタラクティブな探索のためのブラウザベースのClojure replを提供している。

ビルドツール

ビルドツールはClojureライブラリやアプリケーションのビルド、実行、デプロイのための幅広い機能を提供する。最も広く使われているのは Leiningen と Bootだ。
